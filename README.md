# Triple Espresso

Web project coffee shop

Esta página web está separada por header, main y footer. En la sección header se incluye una imágen y elementos de navegación que llevan a otras partes de la página que son recipes, reservation y form, contenidas en main, y el footer.

El formulario contenido en reservation sirve para que los usuarios planeen una visita a la cafetería en una hora y día específicos para un número limitado de comensales.

Sería bueno que el proyecto incluyera un section donde se detallara el menú de la cafetería junto con los precios de los productos para que quienes tengan intención de reservar tengan una idea más clara de qué van a consumir.
